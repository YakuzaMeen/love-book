# 📕 Libro de Amor - 200 Razones Para Amarte

Un proyecto web estático y hermoso que presenta 200 razones para amar a una persona especial, culminando con una propuesta romántica.

## 🎯 Características

- ✨ **200 Razones dinámicas** generadas automáticamente desde JavaScript
- 📱 **Totalmente responsive** - Funciona perfectamente en mobile, tablet y desktop
- 💅 **Diseño romántico** con colores pastel y tipografía elegante
- 🎨 **Slider horizontal** con Swiper.js vía CDN
- 💔 **Página especial** con la razón más importante
- 💍 **Propuesta romántica** con animación de corazones flotantes
- 🚀 **Listo para GitHub Pages** - Sin configuración adicional necesaria
- ✅ **Sin dependencias backend** - Solo HTML, CSS y JavaScript puro

## 📁 Estructura de Archivos

```
love-book/

├── style.css       # Estilos y diseño responsivo

```

## 🚀 Cómo Usar

### Localmente

1. Descarga o clona este repositorio
2. Abre `index.html` en tu navegador
3. ¡Disfruta navegando por las razones!

## 🎨 Personalización

### Cambiar el texto de la propuesta

Abre `script.js` y modifica la variable `proposalText` para personalizar la pregunta (por defecto es "¿Me permites a mí ser el amor de tu vida?") o ajustar la intención.

La propuesta está bloqueada por un flujo de confirmaciones (accesible mediante un candado 🔒) que quien la abra debe confirmar antes de ver el mensaje final.

### Modificar colores

En `style.css`, busca y modifica los colores principales:
- `#ff69b4` - Rosa principal
- `#ff1493` - Rosa oscuro
- `#ffc0cb` - Rosa pastel

### Cambiar fuentes

En `index.html`, modifica la importación de Google Fonts:

```html
<link href="https://fonts.googleapis.com/css2?family=TuFuente:wght@400;700&display=swap" rel="stylesheet">
```

## 📖 Contenido

El proyecto incluye:

- **200 razones para amar** - Un array de 200 strings con diferentes razones
- **Razón 201** - Una razón especial adicional más emotiva
- **Propuesta** - Una página final con dos botones y animación de respuesta
