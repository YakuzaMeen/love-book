# 📖 Libro de Amor - Proyecto Completo ✨

## 🎯 ¿Qué es este proyecto?

Un libro digital romántico interactivo con **200 razones para amar a alguien**, una página especial adicional, y una propuesta romántica con animaciones hermosas. Perfecto para expresar tus sentimientos de manera única y memorable.

## ❓ Preguntas Frecuentes

### ¿Necesito Node.js o NPM?
No, este proyecto es completamente independiente. Solo necesitas un navegador.

### ¿Puedo cambiar el contenido?
Sí, absolutamente. Cada texto es personalizable. Lee [PERSONALIZATION.md](PERSONALIZATION.md).

### ¿Funciona offline?
Sí, excepto por las fuentes de Google que se descargan. El resto funciona completamente sin internet.

### ¿Es seguro para GitHub Pages?
Totalmente. No hay backend, no hay datos sensibles, solo HTML/CSS/JS estático.

### ¿Puedo tener más de 200 razones?
Sí, simplemente agrega más elementos al array `reasons` en script.js.

### ¿Puedo cambiar los colores?
Sí, lee [PERSONALIZATION.md](PERSONALIZATION.md) para una guía completa.

### ¿Puedo compartir este código?
Sí, es código abierto. Hazlo tuyo.
